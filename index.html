INTRODUCIÓN.

Como proyecto final del curso de Microprocesadores se decidió implementar un juego llamado Air Hockey pero de forma digital utilizando como componente fundamental un micro controlador ATMEGA328P, además de algunos como flipflops para un registro de desplazamiento, controladores de display, display 7 segmentos y matrices de LEDs controladas con un integrado Max7219.

MATERIALES

Los materiales usados para este proyecto son:

  1. Flip Flop JK, Referencia 74LS73 - Datasheet.
  2. Compuerta NOT, Referencia 74LS04 - Datasheet.
  3. Display 7 Segmentos Ánodo común.
  4. Manejador de display, Referencia 74LS47 - Datasheet.
  5. Arduino UNO R3.
  6. Sensores Ultrasonicos, Refenrica HC-SR04 - Datasheet.
  7. Cables.
  8. Protoboard.
  9. Adaptador 12V, 1A.
  10. Matrices de LEDs.
  11. Controlador de display, Refencia MAX7219 - Datasheet.

FUNCIONAMIENTO.

SENSORES ULTRA SONIDO.

Los sensores HC-SR04 cuentan con cuatro pines, el primero de estos pines es la alimentación VCC. El segundo pin es el Trig o Trigger, al cual se le envía un pulso TTL de 10 micro segundos, el tercer pin corresponde a echo y se configura como entrada para poder medir el tiempo en proporción y obtener la distancia medida, el ultimo pin corresponde a GND y va conectado a la tierra del circuito, el sensor es capaz de medir distancias desde 2cm a 400cm.

REGISTRO DE DESPLAZAMIENTO.

El RegistroJR consta de un arreglo de 8 flip flops conectados en serie para pasar la información atreves de dos pines del micro controlador de los cuales una actúa como el reloj que permite pasar la información por medio de flancos y la otra es la información. Para lograr pasar la información se hace un ciclo for en el software que pasa 8 bits a cada flip flop estos bits son la cuenta de los goles para los dos displays; una ve la información se encuentra en los flip flops se saca en paralelo 4 bits para el primer decodificador y los otros 4 bits para el otro codificador, donde se encarga de transformar el dato de binario a decimal en el display.

DISPLAY 7 SEGMENTOS.

Para mostrar el marcador se implementaron 2 display 7 segmentos de ánodo común los cuales son manejados con el integrado 74LS47, la entrada de cada manejador es el registro de desplazamiento que se implementó con los 8 flip flop y se explico previamente.

CONTROLADOR MAX7219 + MATRIZ DE LEDS

Para el funcionamiento del controlador Max7219 se definen tres pines digitales como salida para el DIN, CS y CLK. Para que los datos ingresen al registro DIN (registro de 16 bits) en primer lugar el pin CS se coloca en bajo, en el pin correspondiente a DIN se escribe un alto o bajo que será desplazado por el reloj 16 veces para así acomodar el registro. Para sacar el controlador de modo shutdown se escribe en el registro el valor (0x0C<<8)|1, además para que el controlador escanee de 0 a 7 se escribe el valor (0x0B<<8)|7, para prender y apagar leds se escribe en el registro el valor correspondiente a las filas y el LED que se desea prender, por ejemplo si se escribe en el registro (0x05<<8)|(1<<3) se enciende el LED 5,4.
